# Fundamentals

The beginner track. If you're starting from zero, this is where you build the base — the **language** you'll write, the **framework** that turns it into a web app, and the **tool** that lets a team work together on the same code. Take them roughly in the order below, at your own pace. There's no rush and no "behind."

Work through Ruby first, then Rails. Git runs alongside both — you'll reach for it the moment you make your first real change, so keep it handy. Once you've been through all three, you're ready to start picking up work from the main program.
