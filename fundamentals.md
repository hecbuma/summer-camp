# Fundamentals

The beginner track. If you're starting from zero, this is the path — the **language** you write, the **framework** that turns it into a web app, the **data** underneath it, the **pages** people see, and the **workflow** a team uses to ship it all.

Take them in order — they build on each other. Git runs alongside everything else — you'll reach for it the moment you make your first real change. There's no deadline and no "behind."
